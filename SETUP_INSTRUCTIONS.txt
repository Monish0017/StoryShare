============================================
  ⚙️ CONFIGURATION REQUIRED
============================================

🔴 CRITICAL: Before running the app, you MUST:

1. Open: backend\.env

2. Replace this line:
   MONGODB_URI=your_mongodb_atlas_connection_string_here

   With your actual MongoDB Atlas URI:
   MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/storysharing?retryWrites=true&w=majority

3. Change the JWT_SECRET to something secure:
   JWT_SECRET=your_random_secure_secret_key_here

============================================
  🎯 HOW TO RUN
============================================

STEP 1: Open TWO PowerShell terminals in VS Code

STEP 2: In Terminal 1 (Backend):
   cd d:\StoryTellingApp\backend
   npm run dev

   ✅ You should see: "✅ Connected to MongoDB Atlas"
   ✅ And: "🚀 Server is running on port 5000"

STEP 3: In Terminal 2 (Frontend):
   cd d:\StoryTellingApp\frontend
   npm run dev

   ✅ You should see: "➜ Local: http://localhost:5173/"

STEP 4: Open your browser:
   http://localhost:5173

============================================
  ❓ TROUBLESHOOTING
============================================

🔴 Backend won't start?
   → Check MongoDB URI in backend\.env
   → Make sure port 5000 is free
   → Run: cd backend && npm install

🔴 Frontend won't start?
   → Make sure port 5173 is free
   → Run: cd frontend && npm install

🔴 Can't connect to database?
   → Verify MongoDB Atlas IP whitelist
   → Check connection string format
   → Ensure cluster is running

🔴 API calls failing?
   → Check if backend is running on port 5000
   → Check browser console for errors
   → Verify you're logged in for protected routes
